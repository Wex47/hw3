Autonomic Break-Down Program ( 自律崩壊プログラム , Jiritsu Hōkai Puroguram ) is a technique used by Hermit Mio .

Contents

  • 1 Description
  • 2 Usage
  • 4 Site Navigation

Description

The user activates a hacking program while making contact with a device such as a rifle and breaks the device down into its component pieces, rendering it inoperable.

Usage

Hermit first used this technique in her battle against Fie to destroy his sniper rifle.
