Reverse ( リバース , Ribāsu ) is a Cat Leaper technique used by Rebecca Bluegarden .

Contents

  • 1 Description
  • 2 Usage
  • 4 Site Navigation

Description

The user is able to turn back time by a few minutes, retaining the memories of the events that transpire within that timeframe. Using this knowledge they are able to change the course of events. When the user goes back once, an anchor point is created that prevents them from going farther beyond it. It consumes a lot of energy from the user, damaging their body and creating temporary cracks, making it dangerous to overuse in quick succession.

Usage

Rebecca first uses this technique when her Ether Gear first activates as Leaper, turning back time back to the moment Nino was about to strike her with his Soul Arm and flew backward to avoid it in time.

Rebecca uses this technique again when her Cat Leaper activates to turn back time back to the moment Britney catches up to her in the town and follows her to the electronics shop Elecloth once more.

Rebecca uses this technique again to test Cat Leaper's limits twice with Pino 's help and suffers temporary cracks as a result before naming the technique.
