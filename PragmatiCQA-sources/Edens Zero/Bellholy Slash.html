Bellholy Slash ( ベルホーリースラッシュ , Beruhōrī Surasshu ) is a Cat Leaper technique.

Description

The user uses their Ether -charged leg to kick their opponent, sending them flying backward. The name derives from an ability used by the title character of " Wibble Wobble Ruby Bobble " to free a man from another villain's brainwashing.

Usage

Rebecca, upon discovering her Ether Gear , first used this technique to defeat Nino , impressing him by using the manga version of the name instead of the anime version which he preferably enjoys.

Trivia

  • The attack's name is similar to the name of Bell Holy , a sword used by Ruby from Rave Master .
