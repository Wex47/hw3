Ignition Floor ( イグニッションフロア , Igunisshon Furoa ) is an Alchemist technique used by Drakken Joe .

Contents

  • 1 Description
  • 2 Usage
  • 4 Site Navigation

Description

The user creates a mark branching away from them. They then proceed to change the floor at the end of the mark into an explosion.

Usage

Drakken first used this technique to attack Shiki , only for the latter to dodge it.
