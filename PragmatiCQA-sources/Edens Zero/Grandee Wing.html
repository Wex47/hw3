Grandee Wing ( グランディウィング , Gurandi Wingu ) is a Machina Maker technique used in conjunction with the Arsenal Suit .

Description

The user reconfigures the forearm of the Arsenal Suit, creating a large winged blade strong enough to slash an O-Tech in half.

Usage

Weisz first uses this technique to cut Müller in half but is caught off guard by the psychopathic scientist's electric tendril attack, resulting the destruction of the Arsenal Suit.

Trivia

  • The attack's name is a reference to the Sky Dragon Grandeeney from Hiro Mashima 's previous work, Fairy Tail .
