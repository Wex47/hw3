The Enormous Satellite Server ( 巨大衛星サーバー , Kyodaieisei Sābā ) is a gigantic satellite that orbited around Foresta the Verdant Planet.

Description

The Enormous Satellite Server is cylindrical in shape. On the front end of the satellite is a large floating sphere which can be assumed is the part of the server that transmits the data within the server to any receivers on the planet below. On the back end of the satellite, a number of fin-like plates are attached, protruding outwards, and surrounding the Satellite Server is a number of floating plates.

The satellite itself is large enough to be a space station that people can land on if they have a spaceship. The inside of the satellite is made up of various hallways and a single main room where anyone can maintain or control the server from.

History

Ziggy set up the server above the planet of Foresta to distribute a virus that would turn any machine on Foresta hostile to humans. He also used it to hack into everything on the planet and communicate with everyone to deliver the message "Robots will rule over humans" . Müller was put in charge of managing the server by Ziggy.
