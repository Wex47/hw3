Ice Wall ( アイスウォール , Aisu Wōru ) is an ability that utilizes the element of ice.

Description

Ice Wall allows the user to control and manipulate ice, allowing them to to create a enormous intricate barrier made of ice, strong enough to block psychokinetic Ether .
