Flame Bolt ( フレイムボルト , Fureimu Boruto ) is an ability that allows the user to manipulate fire.

Description

Flame Bolt allows the user to control and manipulate fire, allowing them to ignite their arm in flames that would burn everything to cinder and use it in melee combat.
