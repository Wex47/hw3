Heal Atomizer ( ヒールアトマイザー , Hīru Atomaizā ) is a Healing Ether ability that allows the user to mend wounds, undo negative status effects, and repair "living" Machines .

Description

The user collects energy in their palm, and uses it to heal up to multiple targets. This ability does not require direct contact with the victim. If the target is healed too much, the target ends up damaged instead of healed.

Healing can only work on "living" beings, including machines, preventing Sister from using it to repair the Edens Zero after it has been damaged.
