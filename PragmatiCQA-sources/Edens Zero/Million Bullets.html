Million Bullets ( ミリオンバレット , Mirion Baretto ) is a Machina Maker technique.

Contents

  • 1 Description
  • 2 Usage
  • 4 Site Navigation

Description

The user touches the floor and modifies it to create numerous guns; within minutes, that can be activated on their command. The guns can be aimed and fired at a single target in multiple directions.

Usage

Weisz first used the technique to set an ambush for Mosco Versa-0 in the Rogue Out HQ , filling the android with multiple bullet holes as soon as he entered the building.

Weisz in Overdrive uses this technique to defeat Nasseh , finishing him off and destroyed what's left of the Sea Devil .
