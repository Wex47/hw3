Ballistic Simulator (弾道予測シミュレーター Dandō Yosoku Shimyurētā ) is a technique used by Hermit Mio .

Contents

  • 1 Description
  • 2 Usage
  • 4 Site Navigation

Description

The user runs a program to simulate the trajectory of an incoming projectile so that they can react to it and counter the ranged attack.

Usage

Hermit first uses this technique to anticipate the target of Fie 's Flame Bullet so that she could evade it with Aerial Move .
