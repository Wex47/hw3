Prison Matrix (プリズンマトリックス Purizun Matorikkusu ) is a technique used by Hermit Mio .

Contents

  • 1 Description
  • 2 Usage
  • 4 Site Navigation

Description

The user locks down an area causing holographic chains to surround a building signifying that it is sealed.

Usage

Hermit first used this technique against Fie , trapping him in the building he was in so that she could defeat him with her Guns Harvest .
