Red Destiny ( レッドデスティニー , Reddo Desutinī ) is a type of Ether Gear that utilizes strings made of Ether .

Description

This Ether Gear allows the user to create red strings made of Ether that, when touched by someone, binds the victim to the person closest to them and causes them to immediately fall in love with each other. However, when the string connecting them breaks, that love will immediately die and change into hatred so powerful it forces the victims to fight each other until one of them is dead. The user can also utilize these strings on a single target as a means of immobilizing them.
