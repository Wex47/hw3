Wormhole ( ワームホール , Wāmuhōru ) is a type of Ether Gear that allows the user to manipulate space.

Description

This Ether Gear allows the user to create wave-like portals and teleport either themselves or items across both short and long distances, such as behind their enemy or to distant planets.
