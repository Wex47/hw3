Happy Blaster Assault Rifle Mode ( ハッピーブラスター AR ( アサルトライフル ) モード , Happī Burasutā Asaruto Raifuru Mōdo ) is a Happy Blasters mode.

Description

The user begins to charge Ether and then uses it to transform into an assault rifle that increases the range and firepower of its Ether bullets.
