Description

The arcade has several attractions such as a strength test machine named "Space Power", an area for playing card games such as poker, and a stage for bunny-eared pole dancers to perform and entertain the audience on stage.

Location

Game Belial is located somewhere within the Belial Gore.

Trivia

  • When Shiki destroyed the strength test machine, the screen above it displayed "ERRER", a misspelling of "ERROR".
