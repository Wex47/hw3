Steal Hand ( スティールハンド , Sutīru Hando ) is an Ether Gear that enhances thievery.

Description

By using Ether Gear , the user has the ability to successfully steal objects from another person without them noticing by moving the object 10cm. It can also teleport things from other people's hands.
