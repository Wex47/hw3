Spatial Magic: Independent Cube (空間魔法・ 無干渉領域 ( インディペンデント・キューブ ) , Indipendento Kyūbu ) is a high level spatial magic which involves using space to trap individuals.

Overview

This spell allows the user to cause a given area to be cut off in space from the rest of the world. Known to be a high level spell, it has been speculated that it can only be held for an hour, at best; however with an actual practitioner, it lasts less then 40 minutes.

This spell requires the user to stand with their arms spread apart and the usage of Calculation Spells to properly position the " lockbox " made up from space. Once a barrier of space is formed, a "key" is materialized and inserted into the "key hole". The user then turns the "key", locking individuals out. The time it takes to cast the spell is lengthy.

Trivia

  • Although the katakana is "Independent Cube" in English, the kanji given mean "Area of Non-Interference".
