Vector Arrow x3 (ベクトルアロー x3 ( バイスリー ) , Bekutoru Arō Bai Surī ) is a more powerful variation of the magical attack, Vector Arrow .

Overview

Acting similarly to Vector Arrow , the amount of "arrows" (or "vectors") created is triple the initial amount that the original magic is capable of producing. This magic is perfect for holding back multiple enemies and is considerably more powerful though it possess a weakness: it is possible to dodge the magical attack by dodging from the center if the target manages to move fast enough.
