Vector Conduct (ベクトルコンダクト, Vectoru Condacuto ) is a spell in which deals in navigation.

Overview

Requiring users to place invisible markers before hand with Magic, the user then touches the forehead to use their Magic to allow the person in which they choose see the invisible arrows as it guides them to an intended destination.
