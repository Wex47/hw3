Overview

A demon book is a magically imbued book in which uses the properties of a book combined with magic to give it the ability to perform magical feats. It is a known weapon for both sorcerers and witches alike.

Demon Book's Text

Magical feats originating from a Demon book seems influence by a "writing system" that allows the user to be able to perform magical feats. Some of these feats include summoning , create magical restraints , and transport objects/individuals into a separate dimension.
