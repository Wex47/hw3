Lock (錠前, Joumae ) is a Demon Tool used by Arachnophobia in it's magic-based security system.

Appearance

Like its name implies, the Demon Tool resembles a giant rectangular padlock with a large keyhole in it. A glowing light is seen through the keyhole and many pipes and wires connect the Lock to the building in which it is protecting.

Abilities

Lock is capable of creating a powerful, magic barrier that forbids entry in specific parts of a location in which is built to be protected.
