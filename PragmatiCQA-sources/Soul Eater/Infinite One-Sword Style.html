Overview

The Infinite One-Sword Stlyle allows for a user to adapt to both long distance and short distance combat. This style doesn't follow pre-established schemes and the user's katanas seemingly act by free will. The katanas used within this style of fighting aren't normal ones, as each sword has a individual power.
