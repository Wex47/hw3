A Given Name (or birth name), is one of The Three Names that every person has. A Given Name name is the name you are given at birth and has some power over you, although, not to the degree that one's unsealed True Name does. Your Given Name, provided that you have not taken a name (a Taken Name is one of The Three Names ) to seal it, can allow some sorcerers to have physical control over you.  An example is when China Sorrows knew Stephanie's name, and managed to control her. Stephanie broke the spell by giving herself the taken name Valkyrie Cain .

Examples of Given Names:

  • Stephanie Edgley
  • Gerald
  • Paul Matthews
  • Charles Grantham
  • Patrick "Paddy" Hanratty
  • Oisin Caddock
