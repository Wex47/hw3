Usage

First, the user runs towards the person with the possession of the ball. While running, the user then shines a purple light, which turns the surroundings into an outer space background. The user continues to run towards the person. Then, the user accelerates ahead, and a blue/purple light shines at the spot where the opponent is, turns into an X-like shape, and then into fire, still in the X-like shape. They are then trapped in an intersecting inferno.
