A LEGO Set is a building kit that contains numerous bricks and other elements to assemble one or more objects. Some sets may be assembled in various ways to allow the assembly of different models using bricks from one set. All bricks are compatible with bricks from other sets. All sets are categorized in themes like Orient Expedition or Star Wars , under LEGO System . See List of themes .
