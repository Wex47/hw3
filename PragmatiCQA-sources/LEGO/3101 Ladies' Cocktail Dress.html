3101 Ladies' Cocktail Dress is a 1998 Scala set.

Description

The set consists of a suit of clothes suitable for attending parties in. The main portion of the set is a long (shoulder-length) white dress. A filmy green skirt is attached to the lower section of the dress, around shin-height. An orange jacket is set over the dress. A turquoise watch is attached to one of the arms of the jacket, a large string bow is tied behind it, and a yellow clothes hanger is stuck into it.
