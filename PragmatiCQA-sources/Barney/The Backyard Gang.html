The Backyard Gang was a group of children who starred in the Barney & The Backyard Gang series that originally ran from 1988 - 1991 .

Members

The Backyard Gang consists of:

  • Michael ( Brian Eppes )
  • Amy ( Becky Swonke )
  • Tina ( Jessica Zucha )
  • Luci ( Leah Montes )
  • Jason ( Salim Grant ) (First 3 videos only)
  • Adam ( Alexander Jhin ) (7 videos only; absent in Campfire Sing-Along )
  • Derek ( Rickey Carter ) (Last 5 videos)
  • Jeffrey ( Jeffrey Lowe ) (Replaced Adam in Campfire Sing-Along)
