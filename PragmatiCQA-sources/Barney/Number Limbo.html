" Number Limbo " is an original Barney song and a game used on Barney & Friends , which first appeared in the episode " Carnival of Numbers ".

How to Play

Number Limbo is your average game of limbo, except the bar is lowered one number each round. It gets lowered until it gets to one, at which the game ends. A normal game of Number Limbo has 5 rounds, but it has been customized several times throughout the series.
