Essence of Nature
Main appearance(s)
Oracle of Seasons
Use(s)
Obtaining the Huge Maku Seed

The Essences of Nature are eight relics that Link must obtain to help the Maku Tree and save Din in Oracle of Seasons . Each one is hidden in one of the enemy- and puzzle-filled dungeons spread across Holodrum . When Link obtains all eight Essences, the Maku Tree gives him a special seed called the Huge Maku Seed .

List of Essences of Nature

Fertile Soil

  • Location : Gnarled Root Dungeon
  • Guardian : Aquamentus
  • Description : "Seeds scattered across bountiful lands are nourished in this Fertile Soil!"

Gift of Time

  • Location : Snake's Remains
  • Guardian : Dodongo
  • Description : "Seeds sprout as seasons change with the Gift of Time!"

Bright Sun

  • Location : Poison Moth's Lair
  • Guardian : Mothula
  • Description : "Young shoots grow quickly under the warm rays of the Bright Sun!"

Soothing Rain

  • Location : Dancing Dragon Dungeon
  • Guardian : Gohma
  • Description : "Bathed in drops of Soothing Rain, shoots grow into saplings!"

Nurturing Warmth

  • Location : Unicorn's Cave
  • Guardian : Digdogger
  • Description : "Balmy days build strong saplings with their Nurturing Warmth!"

Blowing Wind

  • Location : Ancient Ruins
  • Guardian : Manhandla
  • Description : "Sweet fruit is born when pollen is carried by the Blowing Wind!"

Seed of Life

  • Location : Explorer's Crypt
  • Guardian : Gleeok
  • Description : "Life begins anew when birds carry this, the Seed of Life, to new lands!"

Changing Seasons

  • Location : Sword & Shield Maze
  • Guardian : Medusa Head
  • Description : "Scattered seeds sprout in spring, grow in summer, bear fruit in fall and sleep through winter. It is an endless cycle of life... the Changing Seasons!"
