Infrared ( 赤 ( せき ) 外 ( がい ) 線 ( せん ) , Sekigaisen ) is a Quirk possessed by All For One , who stole it from an unknown person.

Description

Infrared allows the user to sense their entire surroundings using infrared senses.

All For One uses this Quirk to compensate for his blindness and to be able to easily navigate around his surroundings. This Quirk also allows him to sense the emotional state of others to an extent.

Weaknesses

However, the range that he is able to sense is limited. This Quirk can also be disrupted by sensors.
