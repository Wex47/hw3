Jurassic World: Alive is a free-to-play, location-based augmented reality game developed by Ludia. The game was officially released worldwide on May 24, 2018, 20:35 GMT and is available on both iOS and Android.

New Recent Updates

  • Two new bosses have been added into the game; Hydra Boa (a genetic apex Titanoboa) and Haast Maximus (a genetic Haast's Eagle).
  • Other creatures that have been added include Albertosaurus, Argentavis and Deinotherium
  • New hybrids are Andrewtherium (Andrewsarchus+Deinotherium) and Megistocurus (Megistotherium+Doedicurus).
  • Scorpius Rex is now in the game with 3 Gen types.

Trivia

  • This mark the first apparence of arctops, inostrancevia, gorgonops, bajadasaurus, carbonemys, dakotaraptor, coelodonta, hatzegopteryx, purussaurus, scaphognatus, scolosaurus, sonorasaurus, phorusrhacos, kelenken and argentavis in the whole Jurassic Park franchise.
  • Limnoscelis (with a Gen 2) is somewhat show in the code of Jurassic World Alive, implying that the amphibian will appear in an upcoming update of the game.
  • This mark the first time rinchenia appared in a Jurassic Park game, and this mark its second apparence since Jurassic Park Institute.
  • Interestingly, this game share similar hybrids with Jurassic World: The Game . Species that first appeared in JWA like dilophoboa later appeared in this game while some like the megistocurus appeared recently in JWA.
