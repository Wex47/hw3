A planet is a celestial object that orbits a star . Many planets have their own name, but Stargate Command usually assigns designations, which are based on binary codes for the use of extrapolation. Milky Way planets are usually designated with P??-??? , and Pegasus planets are usually designated with M??-??? . There are some exceptions to this however, so it can be presumed that the first letter can be anything as well. (Example: KS7-535 ). This would give a total number of 3 010 936 384 possible designations. This number is still increased because some early designations had fewer letters/numbers, and it implies that the system has been changed after its introduction.
