Pacifier Spring (named by Viper) is located in an unexplored region of France called Jura. It can strengthen the light of an Arcobaleno Pacifier in an instant. It can make a message written in the Arcobaleno Secret Code legible if the message has been damaged. The spring was named by Mammon .
