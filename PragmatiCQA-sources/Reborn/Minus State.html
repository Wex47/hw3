Minus State

Ability information

Users

Tsunayoshi Sawada

Behind the scenes information

Anime Debut

Episode 60
Minus State is a technique which allows the user to neutralize damage by being in an inverted Dying Will Mode . To use this, the user must jump between a positive state and the zero point to discover the timing for inverting.

Trivia

  • This technique was mistaken for the Zero Point Breakthrough: First Edition .
