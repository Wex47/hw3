The Well's Unknown Abyss ( 不 ( せい ) 知 ( てい ) 井 ( し ) 底 ( らず ) , Seiteishirazu ) is a Ten Shadows Technique extension technique that combines the Nue and Toad shikigami .

  • 1 Description
  • 2 Usage
  • 4 Navigation

Description

By creating the shadows of both Nue and the Toad, Megumi can combine them together to create winged toads to aid him in battle. These toads can fly with increased mobility, as well as use their tongues to restrain opponents. Unlike its parent shikigami, these toads are relatively weaker but can be destroyed without consequence and can be summoned as long as Nue and the Toad are alive.

Usage

Megumi first used this technique to fight against Aoi Todo . Later, he used this technique against Noritoshi Kamo by positioning winged toads around himself to catch manipulated arrows targeted to himself.
