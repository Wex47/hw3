Blood Edge ( 血 ( けつ ) 刃 ( じん ) , Ketsujin ) is a Blood Manipulation technique.

Contents

  • 1 Description
  • 2 Usage
  • 4 Navigation

Description

The user performs a process that involves shaping and strengthening the blood's lethality by rotating it at a high speed.

Usage

Choso uses this technique against Yuji Itadori by forming blood knife. After he wounds Yuji's back with Supernova , he quickly stabs Yuji's foot with this blood knife.
