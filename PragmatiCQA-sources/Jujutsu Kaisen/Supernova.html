Supernova ( 超新星 ( ちょうしんせい ) , Chōshinsei ) is a Blood Manipulation attack.

Contents

  • 1 Description
  • 2 Usage
  • 4 Navigation

Description

After using Convergence to create multiple small orbs from compressed blood, the user has the orbs shoot out in every direction like a buckshot.

Usage

Unique to him, Choso invented this technique while he was sealed as a cursed object . He first used this against Yuji Itadori and seriously wounded him. Later, he used this against Naoya Zenin who was surprised to encounter a Blood Manipulation technique he was unfamiliar with.
