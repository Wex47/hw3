Aggressiveness Level is a classification that is used to measure how aggressive a creature is.

Levels

Harmless: It is not dangerous to humans.

Very Low: It is harmless if not disturbed.

Low: It will attempt to scare if threatened.

Medium: It could attack to hurt if threatened or confused. Will attack if attacked.

High: It will attack and even kill if threatened or confused.

Very High: It will attack to kill if something gets into its territory.

Extreme: It will kill for no apparent reason.
