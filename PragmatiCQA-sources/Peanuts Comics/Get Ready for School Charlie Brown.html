Get Ready For School, Charlie Brown! is a 1995 PC-ROM game which features characters from the Peanuts comic strip by Charles M. Schulz .

It is an educational game with many minigames and puzzles, including writing and dressing up Charlie Brown . There are two settings; learn or play.

This game runs on Windows 3.1 and has a reminder at the start to show more educational games by Virgin Sound & Vision.

Voice cast

  • Steven Hartman as Charlie Brown
  • Anthony Burch as Linus
  • Jamie Cronin as Lucy
  • Danielle Keaton as Sally
  • Brittan Reese as Peppermint Patty
  • Bill Melendez as Snoopy and Woodstock

PC requirements

Although the game was released in 1995, it only works on Windows operating systems from 3.1 to 98.
